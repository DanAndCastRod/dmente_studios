# Contexto del Proyecto: D'mente Estudio - Landing Page
**Descripción:** Desarrollo de una landing page de alta conversión para D'mente Estudio, un espacio híbrido en Pereira que ofrece servicios de producción musical profesional y alquiler de salas de ensayo. 

**Objetivo de la UI/UX:** Transmitir una estética inmersiva, técnica y ligeramente disruptiva ("hipnosis del sonido"). La interfaz debe sentirse como un software de producción (DAW) moderno: oscura, con acentos de neón brillante, limpia pero con mucha energía. Debe atraer tanto a productores meticulosos como a bandas (especialmente de géneros con energía en vivo como rock, grunge o urbano) que buscan un lugar para explotar su sonido.

**Estructura de Datos para el IDE:**
- **Framework sugerido:** React / Next.js (o el estándar del entorno).
- **Estilos:** Tailwind CSS (preferible para manejar variables de colores oscuros y neón).
- **Tono del Copy:** Directo, de "músico a músico", profesional pero rebelde.
