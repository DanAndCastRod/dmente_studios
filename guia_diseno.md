# Design System & UI Tokens: D'mente Estudio

## Paleta de Colores (Tailwind Config)
- `primary-neon`: #5B2C6F (Púrpura D'mente - Úsalo para botones principales, hover states y brillos/glow effects).
- `bg-abyss`: #0A0A0A (Negro Abisal - Fondo principal de toda la landing page).
- `surface-dark`: #1A1A1A (Gris Oscuro - Para tarjetas de servicios, modales o secciones destacadas).
- `accent-cyan`: #00FFFF (Cian Eléctrico - Para notificaciones, badges de "Disponible" o pequeñas llamadas a la acción).
- `text-main`: #F5F5F5 (Blanco humo para alta legibilidad).

## Tipografía
- **Headings (H1, H2, H3):** Fuente *Display* extendida (ej. `font-sans`, usando Monument Extended o Syncopate). Pesada, en mayúsculas para títulos principales.
- **Body Text (P, spans):** Fuente geométrica limpia (ej. Montserrat o Inter).

## Extracción de Assets (De las imágenes de referencia)
1. **Logo Principal:** Rostro hipnótico en púrpura con audífonos. (Uso: Navbar izquierdo y Footer).
2. **Asset Visual 1 (Producción):** Tomas de las pantallas con el DAW (Pro Tools/Ableton) y la mezcladora. (Uso: Fondo con overlay oscuro para la sección "Estudio de Grabación").
3. **Asset Visual 2 (Ensayos):** Toma del baterista en vivo con iluminación fucsia/púrpura. (Uso: Imagen principal para la tarjeta/sección de "Alquiler de Sala de Ensayo").
